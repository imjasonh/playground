01 / THE CONSTRAINT

No chords. No audio files. Four monophonic voices.

The NES can make five kinds of sound, but this track leaves its sample channel unused. Every pitched channel can sound only one note at a time. Complexity has to emerge horizontally: rapid arpeggios, alternating registers and lines that imply harmony together.

03 / THE FORM

The first trip lasts 32 bars. The loop remembers only 24.

Each channel has its own backward jump, but all four jumps cover the same musical duration. On the first pass the eight-bar introduction is heard; at 0:42.7 every stream jumps to measure 9.

04 / THE FORMAT

A note and its duration usually share one byte.

The lower five bits select pitch from a movable lookup-table window. Other bits select one of a handful of lengths. A zero pitch means silence. Bytes beginning with zero are control instructions.

CC
110 01100
duration → 32 pitch → G
G2 · half note

The operation names are descriptive names from a reverse-engineered disassembly, not Capcom’s original symbols. The behavior is observable; the original names are lost.

Octave convention: the disassembly labels pulse notes by their timer-table octave. An annotated G2 actually leaves the NES pulse channel at about 196 Hz—concert-pitch G3. Triangle uses a divider twice as large and an octave-higher table base, so its annotated octave already matches concert pitch.

05 / AUTHORSHIP CLUES

It looks hand-arranged because it was.

“We had to translate the notes one by one into machine language.”

Takashi Tateishi, composer

Tateishi described Bubble Man as piano-oriented: an underwater level suggested quietness, but the stage music still needed energy. The ROM’s rolling broken chords resemble keyboard figuration more than guitar block chords.

He composed on instruments, prepared notation, and worked with sound programmer Yoshihiro Sakaguchi’s driver. Explicit note runs, carefully placed jumps and hand-selected duty cycles remain visible in the finished data.
