Web Push demo

A browser front-end for the Web Push backend (a Rust Cloudflare Worker). Point it at your deployed Worker, subscribe, and send a notification to see the full round-trip work.

1 Connect to your Worker

The base URL of your deployed Worker, e.g. https://web-push-worker.your-name.workers.dev. It's saved in this browser and can be shared via ?api=.
