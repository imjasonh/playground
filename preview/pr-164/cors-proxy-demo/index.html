CORS proxy demo

A browser playground for the cors-proxy Cloudflare Worker. Point it at your deployed proxy, send a request to any public API, and read the response the browser would otherwise block.

Read this first

A CORS proxy can read and modify everything passing through it. Only use it for public, read-only data. Never send secrets, cookies, or authenticated requests through a shared proxy, and never execute (eval) content you fetch through one.
