# Example printability reports

Generated by `./generate-examples.sh` with `--cell 4`.

Tree supports use **shared trunks**, **needle tips**, and a simplified
**beam/column physics** model that splits overloaded trunks and thickens
shafts for print stability while keeping contacts snappable.
